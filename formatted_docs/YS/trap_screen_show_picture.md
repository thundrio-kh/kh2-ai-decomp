## trap_screen_show_picture* (visual)

`syscall 0, 74 ; trap_screen_show_picture (2 in, 0 out)`

Shows a picture on the screen

#### Input parameters
| Name | Type | Description
|------|------|------------
| datafile   | string   | filename?
| number   | int   | ID of picture to show
