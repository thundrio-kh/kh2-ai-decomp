## trap_obj_set_stop_timer* (unused)

`syscall 10, 11 ; trap_obj_set_stop_timer (3 in, 0 out)`

Stops the timer

#### Input parameters
| Name | Type | Description
|------|------|------------
| obj   | YS::OBJ_125   | An object
| id   | int   | Timer ID
| time   | float   | Time to stop at
